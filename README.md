## Search

[![Pagefind](https://img.shields.io/badge/Pagefind-2B6CB0?style=flat-square)](https://pagefind.app/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat-square)](LICENSE.md)

- это экспериментальная поисковая система для экосистемы ИТМО, построенная на принципах федеративности.

Главная архитектурная идея — создать единый поисковый слой поверх множества независимых источников, не прибегая к их централизованному хранению. Система выступает как легковесный агрегатор, который объединяет распределенные знания, документы и сервисы в единую точку входа.

## Проект находится на ранней стадии.

На данном этапе поиск работает поверх открытых студенческих GitHub-репозиториев, формируя быстрый полнотекстовый индекс.

### Сейчас поддерживается:

* поиск по лабораторным работам по физике (формат поиска: номер лабораторной, например `1.02`)

### В будущем планируется расширение:

* другие предметы, дисциплины и иные аффилированные знания
* больше типов источников (документы, сайты, конспекты)
* более «федеративная» модель индексации распределённых данных
